# 映画チケット料金システム 仕様
## 前提
- 当日会場で購入するときの処理とする
- 映画→上映日時→枚数→割引の順番で選択するものとする
- 単一映画館が前提とする
- 特別興行の場合は考慮しない
- 割引の併用はできない より大きい割引が適用される
## 映画の種類(料金)
- 通常(1800)
- 名画(2000) ２本
## 日時の区分
- レイトショー
## 割引形態
### 人
- シニア(-800)
- 学生(-800)
- ハンディーキャップ(半額)
### 人と日時
- 女性　水曜(-300)
### 日時
- firstデー(-700)
## アクター
- チケット購入者
## 機能一覧
- 映画、上映日時、座席、枚数を選択する
- 割引を選択する
- 支払い金額を確認する
